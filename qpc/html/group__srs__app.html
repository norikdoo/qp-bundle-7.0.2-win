Preemptive Dual-Mode KernelArchitecture

This section specifies the requirement for a QP Application. These requirements cannot be satisfied by the QP Framework alone, so they are outside the scope of this SRS document. However, these requirements are still important, because they capture the assumptions made in the QP Framework as to how the QP Applications need to operate.

Attention
In case a requirement for the QP Application is not satisfied, QP Framework cannot guarantee the correct execution of the Application.
